📖 Dictionary App


A sleek and interactive Dictionary App that lets you search words and get meanings, synonyms, and pronunciations instantly. Perfect for students, writers, and language enthusiasts.

🌐 Live Demo

[Dictionary App](https://clever-puffpuff-bfd4d0.netlify.app)
✨ Features

✅ Search for any word and get its definition.

✅ View synonyms and pronunciation.

✅ Clean, responsive, and user-friendly UI.

✅ Works completely on the frontend (no backend required).

✅ Instant results using a free dictionary API.

🛠 Technologies Used

HTML5 – structure

CSS3 – styling

JavaScript – functionality

Free Dictionary API – word data
